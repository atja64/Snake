# snake

A version of the classic game [Snake](https://en.wikipedia.org/wiki/Snake_(video_game_genre)) implemented in Java using JavaFX.

## Current Features

* A snake
* An apple
* The snake grows!
* The snake can die :(
* Score counter
* Press p to pause
