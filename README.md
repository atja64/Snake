# Snake

A version of the classic game Snake implemented in Java. Uses the JavaFX canvas to draw graphical elements.

<b>Current Features:</b>

<ul>
  <li>A snake</li>
  <li>An apple</li>
  <li>The snake grows!</li>
  <li>The snake can die :(</li>
  <li>Score counter</li>
  <li>Press p to pause</li>
</ul>
